# Moondrop Dale

Moondrop Dale is a game in which the player is responsible for growing crops, cooking nutritional meals, and completing quests for their fellow villagersâ€”through this process, they learn how to take care of plants and sustain a healthy diet.

This project was built with Unity as a final project for the course *COMPSCI 576 Game Programming* at UMass Amherst.
